# TrueArchitect Dev CLI — Homebrew Tap

Official Homebrew tap for the [TrueArchitect](https://truearchitect.ai) `dev` CLI.

## Installation

```bash
brew tap TrueArchitect-ai/dev
brew install dev
```

## Upgrade

```bash
brew upgrade dev
```

## What is TrueArchitect?

TrueArchitect is an AI-powered development tool that connects to your local codebase and executes tool calls for file operations, code analysis, git, architecture detection, and project understanding.

## Available Formulae

- `dev` — TrueArchitect local development tool executor
